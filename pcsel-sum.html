PCSEL 2nd-Order Grating — Coherent Summation

The same field-summation trick as a Fabry–Perot, only now each "reflection" is a partial wave radiated upward by one unit cell of an in-plane grating. At Bragg (kg·a = 2π) every partial wave arrives in phase along the normal and they add coherently. Detune and they walk off.

Slab + Grating + Partial-Wave Radiation

Guided mode (cyan) travels in-plane. Each grating cell scatters a small partial wave upward. Arrow color encodes the phase of that partial wave's source (cycles as the wave propagates). On-Bragg: all arrows share the same color at any instant → in phase. Off-Bragg: color walks cell-to-cell → phase mismatch.

Phasor Sum (complex plane)

Each cell's partial-wave phasor drawn tip-to-tail. Yellow = resultant. Straight line (all aligned) ⇔ Bragg. Curling back on itself ⇔ detuned, |resultant| → 0. This is exactly your FP multi-layer summation — just with a different phase-per-step.

Far-field Intensity vs Angle

Red = forward guided wave, purple = counter-prop wave (if enabled). On-Bragg: both peak at θ = 0 (single normal beam). Detuned: they split symmetrically. Above the light line (|sin θ| > 1) the partial wave is evanescent — no real-angle emission.

Dispersion & Space Harmonics (Leaky-Wave View)

Normalized Floquet–Brillouin diagram: X = β·a/(2π), Y = a/λ₀. Light cone shaded (fast-wave region, |β| < k₀). Cyan = forward guided-wave harmonics β_m = +neffk₀ + mG; purple = backward harmonics. Solid = fundamental (m=0, always bound); dashed = folded (m=±1). A harmonic radiates whenever its line enters the cone. The emission angle is sin θ = β_m/k₀ = X/Y at the intersection. Green horizontal line: 2nd-order Bragg frequency (a/λ₀ = 1/neff) — the m=−1 forward branch and m=+1 backward branch meet at X=0, which is the open-stopband / PCSEL operating point. Yellow line: current operating frequency.

Tuning Curve: Normal-Direction Intensity vs Period a

Sweep a with neff, λ₀, N fixed. Peak (array factor N²) at Bragg. Width ∝ 1/N — more cells, sharper tuning (higher cavity Q). Side lobes are the familiar sinc-squared fringes.
